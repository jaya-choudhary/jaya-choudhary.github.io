Shrinky Dinks Self-Folding Origami

Overview

This project explores programmable folding using ink patterns on Shrinky Dinks, heat-activated polymers (polystyrene with a pre-stretch). The goal is to design self-folding structures for applications in robotics and space-based deployable systems. This project was inspired by the original work on Shrinky Dinks self-origami done by researchers at North Carolina State University and other studies (Kotikian et al., Science Advances 2017).

Research Goals

  • Investigate fold angle as a function of hinge width and ink color
  • Explore sequential folding using color-coded inks and colored light activation, enabling more complex structures
  • Use robotic plotters like the AxiDraw for precise ink deposition and thermal imaging for actuation timing

Key Takeaways

The project demonstrates how material, geometry, and heat-based actuation can be combined for programmable self-folding structures. This lays groundwork for future soft robotic and deployable space systems.
